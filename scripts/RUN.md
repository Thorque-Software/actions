# Correr scripts vía github

## Install Docker
Wget
```bash
wget -qO- https://github.com/Thorque-Software/actions/main/scripts/install-docker.sh | bash
```

Curl
```bash
curl -fsSL https://github.com/Thorque-Software/actions/main/scripts/install-docker.sh | bash
```
